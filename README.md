Student Score Analysis Project

This project loads a CSV file of student scores, shows the first few rows, calculates basic statistics, and creates two charts (bar chart and pie chart).
It was made as a term project for the course Computing Thinking and SW Coding.

Files Included

myProject.ipynb — main notebook with all the code

scores.csv — input data

scores_barchart.png — bar chart output

scores_piechart.png — pie chart output

What the Program Does

Loads the scores.csv file

Shows the first 5 rows of data

Calculates:

Average score

Maximum score

Minimum score

Median score

Creates:

A bar chart of student scores

A pie chart of score distribution

Saves both charts as PNG files

Includes simple error handling for missing files or invalid data

How to Run

Open myProject.ipynb in Jupyter Notebook or VS Code

Run all cells

The charts will be saved in the project folder

Requirements

Python 3

pandas

numpy

matplotlib

You can install them with:

pip install pandas numpy matplotlib
